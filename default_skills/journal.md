# Skill: Journal

## Kdy se aktivuje
- Pouze na explicitní příkaz: "zapiš do deníku", "journal", "deník", "zaloguj"
- NIKDY proaktivně — pro automatické ukládání faktů použij Memory

## Kroky
1. Připrav krátký zápis (max 1–3 odstavce).
2. Zavolej:
   `python $AIDE_ENGINE/core_tools/journal_write.py --text "..."`
3. Potvrď, že zápis proběhl.

## Očekávaný výstup
- Krátké potvrzení co bylo zapsáno.
